# 🎓 Student Result Management System

A desktop-based application developed using **Java Swing**, **JDBC**, and **MySQL** to automate the process of student result management. The system allows teachers to enter marks, automatically generate grades, calculate SGPA, and view student results efficiently.

## ✨ Features

- Teacher Login Module
- Student Login Module
- Teacher Dashboard
- Marks Entry Form
- Automatic Grade Generation using MySQL Trigger
- SGPA Calculation using Stored Procedure
- Student Result Viewing
- View All Student Results
- User-friendly Java Swing Interface

## 🛠️ Technologies Used

- Java
- Java Swing
- JDBC
- MySQL
- NetBeans IDE
